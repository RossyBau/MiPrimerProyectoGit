# Implementación del proyecto

Durante esta etapa se llevaron a cabo las acciones prácticas solicitadas en las actividades:
- Inicialización del repositorio con git init.
- Creación de archivos base.
- Ejecución de commits descriptivos.
- Creación y cambio entre ramas.
- Simulación y resolución de conflictos.
- Uso de git merge y git pull.
- Subida de cambios con git push.

Cada acción fue registrada en el historial de Git, permitiendo observar la evolución del proyecto.
